Pull Request Hack Day is a hackday who's only output is pull requests to existing projects.

It's happening on Saturday the 27th of Februrary 2016.
(Which just happens to be international Pull Request day.)

We're going to be running a hack day physically located in Birmingham England, but we're hopeful that people can find a way to participate from anywhere!
